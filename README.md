<div align="center">

A ToDo Notes App made with Flutter.

</div>

## Architecture

- Repository for communication with database.
- [BLoC](https://pub.dev/packages/flutter_bloc) for state management.
- [Hive](https://pub.dev/packages/hive_flutter) for data persistence.
